##### 单例模式

核心：保证一个类只有一个实例，并且提供一个访问该实例的全局访问点

常用场景：

1. windows 系统 任务管理器、回收站

2. 网站计数器

3. 配置文件

4. 操作系统的文件系统

5. spring 中的 bean 默认是单例

...

优点: 减少系统开销
